# 字幕质量评测说明

这个功能用于回答“字幕准确率怎么证明”。它用人工参考文本和系统生成字幕做对比，输出 CER、WER、关键词命中率、字幕覆盖缺口等指标。

## 1. 准备参考文本

把人工整理的标准文本放到：

```text
data/reference/
```

示例：

```text
data/reference/input2_reference.txt
```

参考文本可以是纯文本，也可以是人工校对后的 `.srt` 或 `.vtt`。

## 2. 运行评测

```powershell
python tools/evaluate_subtitles.py `
  --candidate data/results/single_test/input2.vtt `
  --reference data/reference/input2_reference.txt `
  --keywords config/custom_keywords.txt `
  --report data/results/single_test/input2_report.json `
  --output-dir data/results/evaluation `
  --basename input2 `
  --db data/results/streamsense.db
```

## 3. 输出文件

```text
data/results/evaluation/input2_eval.json
data/results/evaluation/input2_eval.md
```

如果加了 `--db`，评测结果也会写入：

```text
data/results/streamsense.db
```

## 4. 指标解释

| 指标 | 含义 |
| --- | --- |
| CER | 字符错误率，中文场景最重要，越低越好 |
| WER | 词错误率，中英混合时辅助参考 |
| 关键词命中率 | 配置词表里的领域词在字幕中出现的比例 |
| 补漏后阻塞缺口 | 明显有声但仍没有有效字幕覆盖的时间段数量 |
| 耗时/视频时长 | 小于 1 表示处理速度快于视频播放速度 |

## 5. 答辩说法

可以这样讲：

```text
我没有只靠主观观察字幕效果，而是加入了参考文本评测。
系统可以计算 CER、关键词命中率和补漏后缺口数量，
从文字准确性、领域词识别和时间轴覆盖三个角度评价字幕质量。
```
