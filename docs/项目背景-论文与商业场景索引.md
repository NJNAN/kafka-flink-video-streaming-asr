### 1.3 Whisper 微调与领域适配

- **Whisper-Finetune** (yeyupiaoling, GitHub)：支持 LoRA 微调、无时间戳 / 有时间戳 / 无语音三种数据格式训练，AIShell 测试集微调后 CER 0.031（large-v2）。
  GitHub: https://github.com/yeyupiaoling/Whisper-Finetune
- **Faster Whisper Finetuning** (Vaibhavs10)：8GB 显存消费级显卡微调 Whisper-large-v2，Google Colab 教程。
  GitHub: https://github.com/Vaibhavs10/fast-whisper-finetuning
- **AWS SageMaker 微调 Whisper + LoRA**：100 小时 A100 → LoRA 微调 5-10 次，单 checkpoint 7GB → 60MB。
  Blog: https://aws.amazon.com/blogs/machine-learning/fine-tune-whisper-models-on-amazon-sagemaker-with-lora/
